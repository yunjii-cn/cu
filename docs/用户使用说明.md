# 云集智能文件清理专家 - 用户使用说明

## 简介

云集智能文件清理专家是一款专业的文件管理工具，帮助您快速扫描、分析和清理磁盘中的冗余文件，释放存储空间。

## 安装与启动

### 首次使用（推荐）

1. 下载 `云集智能文件清理专家.exe`（约 8MB 的安装程序）
2. 将其放在您希望作为工作目录的文件夹中
3. 双击运行，安装程序会自动完成：
   - 从 Gitee/GitHub 竞速下载最新稳定版（约 38MB）
   - 自动创建 `ver/` 版本目录并存档
   - 自动将自身替换为程序入口
   - 启动程序
4. 之后双击 `云集智能文件清理专家.exe` 即可使用

> 安装程序会在下载完成后自动替换为程序入口，无需手动操作。首次启动需要网络连接。

### 离线安装

如果无法联网，也可以直接使用完整版 EXE：

1. 下载 `云集智能文件清理专家-v版本号.exe`（约 38MB 的完整程序）
2. 放在目标文件夹中双击运行
3. 程序会自动初始化目录结构并创建入口

### 目录结构

程序运行后的目录结构：

```
工作目录/
├── 云集智能文件清理专家.exe              ← 主入口（双击启动）
├── ver/
│   ├── 云集智能文件清理专家-v版本号.exe  ← 版本存档
│   └── version.json                     ← 版本信息
└── data/                                ← 用户数据
    ├── cleanup_settings.json            ← 设置
    └── scan_results.json                ← 扫描记录
```

## 基本操作

### 1. 扫描文件

1. 在顶部路径栏输入或点击「浏览」选择要扫描的目录
2. 点击「🔍 开始扫描」
3. 扫描过程中可以点击「⏸ 暂停」或「✕ 取消」
4. 扫描完成后，文件列表会按分类显示

### 2. 文件类型筛选

界面顶部的功能设置栏提供多种文件类型筛选：

- **图片文件**：JPG、PNG、GIF、BMP、WebP 等
- **视频文件**：MP4、AVI、MKV、MOV 等
- **音频文件**：MP3、WAV、FLAC、AAC 等
- **文档文件**：PDF、DOC、XLS、PPT 等
- **压缩文件**：ZIP、RAR、7Z 等
- **系统垃圾**：临时文件、缓存、日志等
- **空文件夹**：查找并清理空目录
- **大文件**：查找占用空间过大的文件

点击类型标签展开后，可进一步选择具体的子类型。

### 3. 删除文件

1. 在文件列表中勾选要删除的文件
2. 点击右侧「🗑️ 删除选中」按钮
3. 确认删除操作

### 4. 重复文件检测

1. 点击「🔍 重复检测」按钮
2. 程序会自动扫描选中目录下的重复文件
3. 重复文件会按组显示，方便对比和清理

## 版本管理

### 检查更新

1. 点击左侧导航栏的「软件更新」
2. 点击「🔄 检查更新」按钮
3. 程序会自动从 Gitee/GitHub 获取最新版本信息（竞速机制，哪个先返回就用哪个）

### 版本列表

版本页面提供两种查看模式：

- **版本详情模式**（默认）：卡片式展示，显示完整的版本描述和变更记录
- **版本列表模式**：紧凑列表，适合快速浏览版本号和日期

### 切换版本

在版本列表中，每个版本右侧有操作按钮：

- **● 当前版本**：正在运行的版本
- **切换**：切换到该版本（本地已有该版本 EXE）
- **下载**：从远程下载该版本
- **未提供**：该版本无可用安装包

> 版本切换后程序会自动重启。入口 EXE 会在旧进程退出后自动更新为切换的版本。

### 版本存储机制

- 每个版本是一个完整的独立 EXE 文件（约 38MB），存放在 `ver/` 目录中
- 主入口 `云集智能文件清理专家.exe` 是当前版本 EXE 的硬链接，不额外占用磁盘空间
- 切换版本时，入口 EXE 会自动更新为新版本的硬链接
- 根目录下的旧版本号 EXE 会在下次启动时自动清理

## 常见问题

### 安装程序下载失败

- 检查网络连接是否正常
- 安装程序会依次尝试 Gitee 和 GitHub 两个下载源
- 如果所有源都失败，请使用离线安装方式

### 启动时提示"正在运行"

程序同时只能运行一个实例。如果之前异常退出，请打开任务管理器结束残留进程后重试。

### 版本切换失败

- 确认 `ver/` 目录中有对应版本的 EXE 文件
- 如果是从远程下载的版本，检查网络连接
- 切换需要旧进程完全退出，请等待几秒

### 扫描速度慢

- 首次扫描大量文件可能需要较长时间
- 可以缩小扫描范围，只扫描特定子目录
- 使用文件类型筛选减少扫描量

### 删除文件无法恢复

删除操作是永久性的，删除前请仔细确认。建议对重要文件先备份再删除。
