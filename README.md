## ✨ 功能亮点

| 功能 | 说明 |
|:-----|:-----|
| 🔍 **智能文件扫描与分类** | 全盘扫描或指定目录，按文件类型智能分类，一目了然 |
| 🎨 **文件类型自定义颜色标记** | 为不同文件类型设置颜色标记，快速识别目标文件 |
| 🔁 **重复文件检测** | 智能检测重复文件，支持批量留存策略，一键清理冗余 |
| 📦 **预设规则包** | 内置常用清理规则包，一键应用，无需手动配置 |
| 📊 **分析报告** | 生成文件分析报告，磁盘占用、类型分布尽在掌握 |
| 🗑️ **批量删除/移动/导出** | 支持文件批量删除、移动到指定目录、导出文件列表 |
| 🔄 **版本管理与硬链接切换** | 内置软件更新，多版本共存切换，硬链接映射零延迟 |

### 两种安装方式

**方式一：在线安装（推荐新用户）**

1. 下载 `云集智能文件清理专家-vX.X.X.exe`（约 30MB）
2. 双击运行，程序自动部署目录结构
3. 首次启动即可使用

**方式二：软件内更新（老用户）**

1. 打开已安装的软件
2. 进入「版本更新」页面，检查远程更新
3. 一键下载新版本，自动完成升级

## 📖 使用指南

### 快速上手

1. 启动软件，选择扫描目录或全盘扫描
2. 查看扫描结果，按文件类型、大小等条件筛选
3. 勾选需要清理的文件，一键删除或移动

### 文件分类与标记

- 扫描结果按文件类型自动分类展示
- 支持自定义颜色标记，快速区分不同类型文件
- 可按类型、大小、修改时间等多维度筛选

### 重复文件处理

- 智能检测重复文件，按内容哈希精准匹配
- 提供批量留存策略：保留最新/最旧/最大/最小
- 一键清理冗余副本，释放磁盘空间

### 预设规则包

- 内置常用清理规则（临时文件、缓存文件、日志文件等）
- 支持自定义规则，灵活配置扫描与清理策略
- 一键应用规则包，无需逐项手动设置

### 版本更新

- **版本更新**页面：检查远程更新，一键下载新版本
- 支持多版本共存，可随时切换回旧版本
- 版本切换采用硬链接技术，瞬间完成无需等待
